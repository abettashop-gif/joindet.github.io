ðŸ“Œ Panduan untuk Guru (Aplikasi MyQrAbsensi)

Ikuti 5 langkah mudah ini untuk menyiapkan aplikasi Anda:

  1. Daftar Akun Guru (Registrasi)
    Buka aplikasi, lalu pilih opsi "Daftar Akun Baru". Isi semua data yang diperlukan seperti Nama, Email, dan buat Username serta Password Anda.
  2. Login & Buat Kelas
    Setelah berhasil mendaftar, buka kembali aplikasi dan Login. Masuk ke menu "Manajemen Kelas", lalu tekan tombol + untuk membuat kelas baru (contoh: "10-A").
  3. Input Data Siswa
    Masuk ke kelas yang baru dibuat, pilih "Manajemen Siswa". Di sini Anda bisa menambah data siswa (Nama dan NISN).
  4. Cetak QR Code Siswa
    Jika semua data siswa sudah lengkap, kembali ke menu "Manajemen Kelas", masuk ke detail kelas Anda, lalu pilih "Buat Kartu/QR". Aplikasi akan membuat file yang siap dicetak.

    Sekarang Anda bisa mulai melakukan absensi harian di kelas dengan memindai QR code yang sudah dicetak.

Kebijakan Privasi (Privacy Policy)

Terakhir diperbarui: 21 November 2025

Aplikasi MyQrAbsensi dan Izin Siswa ("kami", "aplikasi") berkomitmen melindungi privasi Anda. Berikut informasi yang kami kumpulkan dan bagaimana kami menggunakannya:

Informasi yang Kami Kumpulkan

  • Aplikasi Guru: Kami menyimpan data Akun Guru, data Siswa & Kelas yang Anda masukkan, dan riwayat absensi. Data ini mayoritas tersimpan di HP Anda. Kami juga mengumpulkan ID Perangkat khusus untuk keperluan aktivasi lisensi.
  • Aplikasi Izin Siswa: Kami mengumpulkan data yang Anda masukkan pada formulir (NISN, Nama Siswa, Jenis Izin, Tanggal) untuk dikirim ke sistem sekolah.

Bagaimana Kami Menggunakan Informasi

  • Untuk Fungsi Aplikasi: Semua data digunakan untuk menjalankan fitur inti aplikasi, seperti rekam absensi, rekapitulasi, dan proses pengajuan izin.
  • Untuk Aktivasi Lisensi (Aplikasi Guru): ID Perangkat digunakan untuk memvalidasi kunci aktivasi.

Keamanan Data

Kami tidak menjual atau membagikan informasi pribadi Anda kepada pihak ketiga tanpa izin, kecuali yang diwajibkan oleh hukum. Data sensitif seperti daftar siswa dan riwayat absen disimpan secara lokal di perangkat Guru.
